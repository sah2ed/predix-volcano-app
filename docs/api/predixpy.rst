
PredixPy: Python SDK
====================

The Predix Python SDK provides client code to interact with Predix Services.
It is available from PyPI::

    pip install predix

The following classes are used as part of the Digital Volcano solution.  Learn
more about all the services provided from the Github page for PredixPy_.

.. _PredixPy: https://github.com/predixpy/predixpy.git
